Que es el desarrollo web?
El desarrollo web es el proceso de crear y mantener sitios y aplicaciones web, 
que abarca desde el diseño de la interfaz de usuario hasta la programación, 
optimización y seguridad. Incluye el desarrollo "frontend" (lo que el usuario 
ve e interactúa) y el "backend" (la lógica del servidor, bases de datos y 
funcionamiento interno), o el "full-stack" que combina ambas áreas. 

Componentes principales
Frontend: Se enfoca en la interfaz de usuario (UI) y la experiencia de usuario (UX), 
asegurando que el sitio sea atractivo y fácil de usar. Los lenguajes comunes son HTML, 
CSS y JavaScript. 
Backend: Se encarga de la lógica y funcionalidad que no es visible para el usuario. 
Esto incluye gestionar la base de datos, la autenticación de usuarios y la lógica del 
sistema utilizando lenguajes como PHP o Python. 
Full-stack: Combina tanto el desarrollo frontend como el backend, lo que permite a un 
desarrollador manejar todos los aspectos de una aplicación web, desde la interfaz hasta el servidor.
