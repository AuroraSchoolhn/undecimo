Los ciclos for en Python son estructuras de control que permiten iterar sobre una secuencia 
(como una lista, tupla, cadena de texto o un rango) o cualquier otro objeto iterable. 
Esto significa que el código dentro del ciclo se ejecutará una vez para cada elemento de 
la secuencia.

Sintaxis básica:

for variable in iterable:
    # Bloque de código a ejecutar en cada iteración


Ejemplo:

frutas = ["manzana", "banana", "cereza"]
for fruta in frutas:
  print(fruta)


for i in range(5):  # Genera números del 0 al 4
  print(i)
