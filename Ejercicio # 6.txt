DEF
En Python, def es una palabra clave que se utiliza para definir una función, 
que es un bloque de código reutilizable diseñado para realizar una tarea específica. 
Para definir una función, se usa def seguido del nombre de la función, paréntesis () y dos puntos :, 
con el código de la función indentado debajo.

def suma(a, b):

    resultado = a + b
    return resultado

# Ejemplo de uso:
num1 = 5
num2 = 3
suma_total = suma(num1, num2)
print(f"La suma de {num1} y {num2} es: {suma_total}")
