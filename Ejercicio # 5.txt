Ciclo while
El ciclo while en Python ejecuta un bloque de código repetidamente mientras una condición sea verdadera. 
Su sintaxis es while <condición>:, seguida por el código indentado que se ejecuta, el cual debe modificar 
la condición para evitar un bucle infinito. Es útil cuando no se sabe de antemano cuántas veces debe 
repetirse una tarea, ya que la ejecución depende de un estado dinámico. 

contador = 0
while contador < 5:
    print(f"El contador es {contador}")
    contador += 1 # Incrementamos el contador para evitar un bucle infinito
